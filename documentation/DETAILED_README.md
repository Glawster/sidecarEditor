# Sidecar Editor

A Qt (PySide6) desktop tool for reviewing and refining image prompt sidecar files generated by LLaVA.

## Features

- Browse images in a folder
- Load and edit `*.prompt.json` sidecar files stored next to images
- Preview the original image and toggle to the generated output image
- Persist UI state and paths using the kohyaConfig.py module
- Automatic backup of sidecar files when saving (`.prompt.json.bak`)

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application with:
```bash
python -m sidecarEditor
```

Or from within the project directory:
```bash
cd /path/to/sidecarEditor
python -m sidecarEditor
```

## First Steps

1. **Set Input Folder**: Use `File > Set Input Folder` to select the folder containing your images
2. **Set Output Folder** (optional): Use `File > Set Output Folder` to specify where generated images are stored
3. **Select an Image**: Click on a thumbnail to view it and edit its sidecar
4. **Edit Prompt**: Modify the prompt and negative prompt in the editor panel
5. **Save**: Click "Save Sidecar" to save your changes

## Sidecar File Format

Sidecar files are JSON files stored next to images with the `.prompt.json` extension:

```
image.png
image.png.prompt.json
```

Example sidecar content:
```json
{
  "prompt": "A beautiful sunset over mountains",
  "negative_prompt": "blur, low quality",
  "tags": [],
  "metadata": {}
}
```

## Configuration

Settings are stored in `~/.config/kohya/kohyaConfig.json` under the `sidecarEditor` key.

## Project Structure

```
sidecarEditor/
├── src/                      # Core logic (no Qt dependencies)
│   ├── sidecarCore.py       # Scan, load/save sidecars
│   ├── outputResolver.py    # Original → output lookup logic
│   └── sidecarConfig.py     # Config wrapper
├── ui/                       # UI layer (Qt only)
│   ├── mainWindow.py        # Main application window
│   └── widgets/
│       ├── thumbnailList.py # Image list widget
│       ├── imagePreview.py  # Image preview widget
│       └── editorPanel.py   # Sidecar editor widget
├── __main__.py              # Entry point
└── requirements.txt
```
